# SAMOS_R

Quarto website for **SAMOS Core Module 3** (Introduction to R + Biostatistics).

## Build locally

```bash
quarto render
```

Preview:

```bash
quarto preview
```

## Deployment

This site is intended to be deployed on Netlify from the GitHub repo `ajsmit/SAMOS_R`.

- Build command: `quarto render`
- Publish directory: `_site`
